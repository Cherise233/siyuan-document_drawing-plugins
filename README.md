# Document Drawing — 文档绘图插件

在思源笔记文档上实现多图层透明绘图，支持桌面端（Windows/macOS/Linux）和平板端（Android/iOS/鸿蒙）。（下载文件放进'工作空间/data/plugins'文件目录下，移动端使用桌面端同步）

## 功能概览

### 绘图工具

| 工具 | 说明 |
|------|------|
| ✏️ 画笔 | 贝塞尔曲线平滑绘制，可调粗细/颜色/透明度 |
| 🖍️ 荧光笔 | 默认 20px 粗细 + 60% 透明度，整笔路径一次性描边无重叠斑点 |
| 🧽 橡皮擦 | 圆形擦除，可调大小 |
| ⬜ 选择 | 框选区域 → 移动 / Delete 删除 |

### 图层管理

- 多图层叠加，支持显示/隐藏/重命名/删除
- 点击图层行可切换当前绘制图层
- 图层数量可为零

### 撤销

- 工具栏按钮 ↩️ 或 **Ctrl+Z** 撤销上一笔
- 最多 30 步历史

### 手势（平板端）

| 手势 | 功能 |
|------|------|
| 手指双击画布 | 切换画笔 ↔ 橡皮 |
| 双指同时点击画布 | 撤销上一笔 |
| 手指拖动 | 滚动页面 |
| 笔书写 | 绘图 |

可在设置中独立开关这两个手势。

### 导出

- **PNG 图片**：桌面端截文档内容 + 图层合并，保存到自定义目录；平板端上传至思源本地服务器 → 浏览器预览/下载
- **PDF 文件**：桌面端同上生成 PDF；平板端构建 PDF Blob 上传后浏览器下载
- 导出时可勾选要合并的图层
- 设置中可自定义导出文件存放目录

### 设置

- 画笔默认大小 / 颜色 / 不透明度
- 荧光笔默认大小 / 不透明度
- HEX / RGB 双格式颜色输入
- 预设颜色集（可自定义添加/删除）
- 手势控制开关
- 导出目录（桌面端）

## 使用方式

### 桌面端

1. 点击顶栏图标打开菜单 → 「添加图层」→ 弹出浮动工具栏
2. 在工具栏选择工具开始绘图
3. 调整颜色（圆点按钮 → 取色弹窗）、粗细、透明度
4. 「图层管理」查看/切换/重命名/删除图层
5. 「导出」→ 勾选图层 → 选择 PNG/PDF → 预览 → 保存

### 平板端

1. 操作和桌面端一致
2. 手指双击画布可切换画笔/橡皮
3. 双指点击画布可撤销
4. 导出后预览弹窗中点击按钮跳转浏览器查看/下载

## 跨平台

- 桌面端通过 Node.js fs 存储图层数据到文件
- 平板端通过 localStorage 存储，避免 WebView 崩溃
- 颜色取色弹窗在平板端极简显示
- 导出自动适配：桌面端写文件，平板端上传 → 跳转浏览器
