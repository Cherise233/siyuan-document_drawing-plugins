# Document Drawing — 文档绘图插件

在思源笔记文档上实现多图层透明绘图，类似 Word 的 Draw 功能。

## 功能

- ✏️ 多透明图层，支持显示/隐藏/重命名/删除
- 🖌️ 画笔 + 橡皮擦工具
- 💾 按文档独立保存图层数据
- 📂 重新打开文档自动恢复图层
- 📄 导出合并为 PNG 图片
- ⚙️ 设置：画笔大小、颜色、自动保存

## 使用

1. 点击右上角 📋 图标打开下拉菜单
2. 「添加图层」创建第一个图层（自动弹出绘图面板）
3. 在面板中选择工具，在画布上绘图
4. 右侧图层面板可管理多个图层

## 开发

基于 SiYuan Plugin API，单文件结构（所有 Manager 内联在 index.js 中）。
